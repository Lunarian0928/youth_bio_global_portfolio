# youth_bio_global

의료 AI 개발자 포지션 지원을 위한 포트폴리오 리포지토리입니다. 독립된 두 개의 서브프로젝트로 구성되어 있습니다.

## 구성

### [`portfolio/`](portfolio/README.md)
React + Vite로 만든 슬라이드형 포트폴리오 웹사이트. 의료영상 AI 도메인 지식 설명과, 직접 구현한 OCT5k Segmentation 프로젝트 소개를 담고 있습니다.

### [`oct_retinal_segmentation/`](oct_retinal_segmentation/README.md)
OCT5k 데이터셋으로 망막 5개 층을 픽셀 단위로 분류하는 U-Net + ResNet-50 Segmentation 프로젝트. EDA, 전처리, Optuna 하이퍼파라미터 탐색, K-Fold 학습, 평가까지 전체 파이프라인을 포함합니다.

각 폴더의 README에서 자세한 내용을 확인할 수 있습니다.
